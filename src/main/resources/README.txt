Selenium Java Maven Project
Descripción
Este proyecto utiliza Selenium y Java con Maven para navegar en una página web y agregar un producto al carrito de compras.

Requisitos
Java 8 o superior
Maven
Selenium
WebDriver compatible con el navegador deseado (Chrome, Firefox, etc.)
Uso
Clona el proyecto en tu equipo local
Abre una terminal en la carpeta del proyecto
Ejecuta el comando mvn compile para compilar el proyecto
Ejecuta el comando mvn test para ejecutar el script de prueba
Advertencias
Asegúrate de tener la última versión del navegador y del WebDriver correspondiente.
Verifica que la URL de la página web esté correcta en el código antes de ejecutar el script.
